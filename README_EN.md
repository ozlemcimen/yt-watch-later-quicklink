# YouTube Mini Menu Shortcut

This extension adds a quick "Watch Later" shortcut to the collapsed (mini) sidebar on the left side of YouTube. It allows you to access your playlist with a single click.

## Features

*   **Mini Menu Integration:** Adds a sleek icon to the mini sidebar (Mini Guide).
*   **Language Support:** Automatically uses "Daha Sonra İzle" (TR) or "Watch Later" (EN) based on your browser language.
*   **Compatible Design:** Fully compatible with both dark and light modes on YouTube.

## Installation

1.  Download this project folder to your computer.
2.  Go to `chrome://extensions/` in your Chrome browser.
3.  Turn on **Developer mode** in the top right corner.
4.  Click the **Load unpacked** button.
5.  Select the project folder.

Once installed, open YouTube (or refresh the page). You will see the new icon appear in the mini sidebar automatically.
